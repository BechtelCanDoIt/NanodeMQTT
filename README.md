NanodeMQTT
==========

Requirements
------------

* NanodeUIP - uIP stack for Nanode - http://github.com/sde1000/NanodeUIP
* NanodeUNIO - MAC address reading for Nanode - http://github.com/sde1000/NanodeUNIO


Limitations
-----------

- Maximum packet size is 127 bytes
- Only QOS 0 supported
